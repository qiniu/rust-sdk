## 概览

基于 Ureq 库提供 HTTP 客户端接口实现，仅提供阻塞接口的实现，不提供异步接口的实现。

## 安装

```toml
[dependencies]
qiniu-ureq = "0.1.3"
```

## 最低支持的 Rust 版本（MSRV）

1.60.0
