## 概览

基于 Isahc 库提供 HTTP 客户端接口实现（同时实现阻塞接口和异步接口，异步实现则需要启用 `async` 功能）

不过由于 Isahc 库本身核心是使用异步接口实现，所以即使不启用 `async` 功能，也会引入异步相关库。

## 安装

### 不启用异步接口

```toml
[dependencies]
qiniu-isahc = "0.2.1"
```

### 启用异步接口

```toml
[dependencies]
qiniu-isahc = { version = "0.2.1", features = ["async"] }
```

## 最低支持的 Rust 版本（MSRV）

1.70.0
